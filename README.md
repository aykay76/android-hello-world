# android-hello-world

Install android-studio from AUR, then allow it to install additional components.
Install android-emulator from AUR, this needs ncurses
Enable developer mode on phone by tapping build number in system settings 7 times
Enable USB debugging
sudo adb kill-server
sudo adb start-server
connect phone - accept RSA key
adb devices - shows list of connected devices
Deploy from Studio to phone
